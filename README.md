**Tecnologias:** PHP • MySQL • HTML • CSS • JavaScript • Git

Sistema web completo para gerenciamento de clientes, técnicos, administradores e Ordens de Serviço, incluindo autenticação de usuários, geração de relatórios em PDF e acompanhamento do status da OS.

## 📖 Sobre o Projeto

O sistema foi criado para facilitar o gerenciamento de clientes, técnicos, administradores e Ordens de Serviço (OS), centralizando todas as informações em uma única plataforma.

Além do gerenciamento interno, o sistema permite que o cliente acompanhe o andamento de sua Ordem de Serviço por meio de uma área dedicada.
